Israel Hiking Project

This map was generated from Open Street Map data. This data is free and can be viewed or edited here. All you need to do is create an account and start mapping.

The map can also be downloaded and used offline on Android devices. You'll need to install OruxMaps and place the two files on your SD card under /sdcard/oruxmaps/mapfiles/IsraelHiking.

Below are some links related to this project

  • Github - where we store all files related to generation of the map and some instructions on how to Do It Yourself.
  • Israel OSM Wiki Project - where you can find some basic rules on how to add data to OSM.
  • Israel OSM Forum - where you can post questions and look for answers.
