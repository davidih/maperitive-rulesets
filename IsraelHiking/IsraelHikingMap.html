Israel Hiking Project

עברית

This map was generated from Open Street Map (OSM) data where it can be be viewed and edited. All you need to do is create an account and start mapping.

The map can also be downloaded and used offline on Android devices. You'll need to install OruxMaps and place the two files on your SD card under
/sdcard/oruxmaps/mapfiles/IsraelHiking.

Below are some links related to this project

  • See Github for files and instructions on how to create the map by yourself
  • See Israel OSM Wiki Project for common rules of the Israeli Open Street Map community
  • See Israel OSM Forum for the Israeli Open Street Map forum where you can post questions and look for answers
